## 🏴 Manifesto

私たちが定義する「Punk」とは、社会への反逆ではありません。
**壊すべきは、昨日までの自分たち自身と、凝り固まった自身の古い価値観です。**

そして「Mad」とは、狂気ではありません。
**今日という二度と来ない瞬間を、誰よりも真剣に生きる熱狂のことです。**

工場で作られたような「正解」に抗い、生まれてきた「歪み（Distortion）」を価値に変える。それがMadPunksです。

## 🌍 Services & Structure

MadPunksは、以下の4つの領域を通じて経済圏と文化圏を構築しています。

| Category | Service Name | Description |
| :--- | :--- | :--- |
| **DIGITAL & TECH** | Platform System | ブランドとコミュニティを守るためのデジタル基盤、SaaS型運営支援システムの構築 |
| **CULTURE** | MadPunk Fes | 循環型フェスの企画・運営、音楽・映像・アートを通じた表現の場の創出 |
| **LIFESTYLE** | The BASE | 飲食（Soul Soup）、宿泊、コワーキングスペースなど「暮らすように働く」拠点の運営 |
| **ECONOMY** | Supply & Support | 規格外野菜等の直売所運営、経営コンサルティング、財団機能の整備 |

## 🗺 Roadmap

* **Phase 0: GENESIS** - 菜園（Saien）の開拓とプロトタイプ開発
* **Phase 1: EXPANSION** - リアル拠点（The BASE）の建設とコミュニティ醸成
* **Phase 2: ESTABLISHMENT** - 経済圏の確立と循環システムの完成
* **Phase 3: REVOLUTION** - 地域と世界を繋ぐ文化圏の創造

## 🚀 Local Development

開発環境でサイトを確認する手順：

1.  **Clone the repo**
    ```bash
    git clone https://github.com/MadPunks-Official/madpunks.github.io.git
    ```
2.  **Run locally**
    VS Codeの [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 拡張機能を使用するか、Pythonなどでサーバーを立ち上げてください。
    ```bash
    # Python 3 example
    python -m http.server 8000
    ```
3.  ブラウザで `http://localhost:8000` にアクセス。
